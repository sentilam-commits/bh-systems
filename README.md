# BH Systems – Landing v2

Landing page estática para BH Systems.

## Estructura

```text
bh-systems-site-v2/
├── index.html
├── styles.css
├── script.js
├── robots.txt
├── sitemap.xml
├── vercel.json
└── assets/
    ├── favicon.svg
    ├── branding/
    │   └── bh-systems-logo.png
    └── demo/
        ├── puntos-de-venta-desktop.png
        ├── registrar-movimiento-mobile.png
        ├── puntos-de-venta-mobile.png
        └── facturacion-desktop.png
```

## Demo

Todos los botones de demo apuntan a:

```text
https://demo.bhsystems.co
```

## Ejecutar localmente

```bash
cd bh-systems-site-v2
python3 -m http.server 8080
```

Luego abre:

```text
http://localhost:8080
```
